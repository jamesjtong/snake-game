Snake App

**To Start**  
1. yarn install 
2. npm start 
3. visit localhost:8080 in your browser 

**Features**  
-Score  
-Pause/Play  
-Speed Increase/Decrease  
-Movement via arrow keys or vim-directions (hjkl)
